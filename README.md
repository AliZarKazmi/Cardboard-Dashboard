## Tech Stack

**Frontend:** HTML5 / CSS3 / ReactJS

## Quick start

Clone the repo

```bash
  git clone https://github.com/blaiti/React-Dashboard
```

Install React-Dashboard with npm

```bash
  cd React-Dashboard
  npm install
```

## Run Locally

To run locally, run the following command

```bash
  npm start
```

## Deployment

To create a production build

```bash
  npm run build
```
